# 🚑 Medox – AI-Powered Healthcare Platform

> **"Smart Healthcare. Instant Care. Anytime, Anywhere."**

Medox is a full-stack AI-powered healthcare platform designed to simplify medical access, enhance patient experience, and provide intelligent assistance through modern web technologies.

---

## 🌟 Features

✨ **AI Medical Assistant**

* Smart chatbot for basic health queries
* Instant responses powered by AI

👨‍⚕️ **Doctor Booking System**

* Browse available doctors
* Book appointments seamlessly

🚨 **Emergency Services**

* Quick access to emergency help
* Real-time assistance features

📊 **Patient Dashboard**

* Manage appointments
* Track health interactions

💳 **Integrated Payment System**

* Secure payments (UPI, Card, etc.)
* Smooth checkout experience

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Tailwind CSS / Custom UI

### Backend

* Node.js
* Express.js

### Database

* MongoDB

### Other Tools

* REST APIs
* AI Integration (OpenAI / LLMs)
* Razorpay (Payments)

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/andrew-garcia7/medcare-ai.git
cd medcare-ai
```

---

### 2️⃣ Install dependencies

#### Frontend

```
cd frontend
npm install
```

#### Backend

```
cd backend
npm install
```

---

### 3️⃣ Run the project

#### Start Backend

```
npm start
```

#### Start Frontend

```
npm run dev
```
